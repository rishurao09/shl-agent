# SHL Assessment Recommendation System

## Overview

This project is a Python-based recommendation system that suggests relevant SHL assessments based on user input (job role or skill).

## Features

* Keyword-based search and recommendation
* Smart ranking using weighted scoring
* Handles multiple keywords
* Basic typo correction using fuzzy matching
* Interactive web interface using Streamlit

## How it Works

The system processes user input to extract meaningful keywords and matches them against assessment names and descriptions. Each match is scored, and results are ranked based on relevance.

## How to Run

### 1. Install dependencies

pip install streamlit

### 2. Run the application

streamlit run app.py
